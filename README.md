# рџљЂ Getting Started

## 1. Clone Repository

```bash
git clone https://github.com/Sreevanth9/sreevanth-portfolio.git
```

## 2. Navigate to Project

```bash
cd sreevanth-portfolio
```

## 3. Install Dependencies

```bash
npm install
```

## 4. Run Development Server

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
